# Ada LaLola 🌭

A modern single-page application website for Ada LaLola, a beloved hole-in-the-wall hotdog stand serving up authentic street food since day one.

## About

Ada LaLola is your go-to spot for the best hotdogs in town. From classic franks to gourmet creations, we've got the perfect combination of toppings, sauces, and love in every bite.

This website showcases our menu, location, hours, and lets customers connect with us online.

## Features

- **Menu Display** - Browse all our hotdog varieties and toppings
- **Location & Hours** - Find us and check when we're open
- **Contact Information** - Get in touch with us
- **Responsive Design** - Works seamlessly on all devices
- **Fast & Lightweight** - Single-page app built for speed

## Tech Stack

- Frontend framework (to be determined)
- HTML5 & CSS3
- JavaScript/ES6+
- Responsive design with modern web standards

## Getting Started

### Prerequisites

- Node.js (v14+)
- npm or yarn

### Installation

```bash
git clone https://github.com/yourusername/ada-lalola.git
cd ada-lalola
npm install
```

### Development

```bash
npm start
```

Opens the application in your browser at `http://localhost:3000`

### Build

```bash
npm run build
```

Generates optimized production build.
